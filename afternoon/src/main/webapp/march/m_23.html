글꼴 관련 스타일

font(글꼴) = 크기, 굵기, 글꼴형태를 지정할 수 있다.

font-weight 폰트의 굵기 : normal, bold, border
font-style 기울기 : normal, itailc
font-size 크기 : 보통 px로 지정한다.

텍스트 스타일

글꼴 자체가 아닌 글자들(텍스트들)에 대한 스타일을 말한다.

  • text-shadow:그림자효과
  • white-space: 공백문자(스페이스바), 탭, 줄바꿈에 대한 처리
    • 값 공백 (여러)줄 줄바꿈
    normal(기본값) 한개로 합쳐져 보임 합쳐져 한줄로 표시 자동
    nowrap 한개로 합쳐져 보임 합쳐져 한줄로 표시 자동 안됨
    pre 유지 유지안됨 자동 안됨
    pre-wrap 유지 유지 자동
    pre-line 한개로 합쳐져 보임 유지 자동

    기타특징
    마우스를 컨텐츠에 갖다 올렸을 때 보이게 처리하는 법 : {overflow: visible;}
    말줄임표로 표시하는 방법 : text-overflow: ellipsis;
    글씨가 더 많을 때 숨기는 법 : { overflow:hidden; }

(pre-wrap)문 대통령과 윤 당선인은 내일 오후 6시 청와대 상춘재에서 만찬을 겸해 만날 예정입니다.
